# 📝 To-Do-List-Apps-0.2

Aplikasi pengelola rutinitas harian sederhana namun fungsional yang dibangun dengan **HTML5, CSS3, dan JavaScript Vanilla**. Aplikasi ini dirancang untuk membantu pengguna mencatat tugas harian dengan fitur penyimpanan otomatis.

---

## 🚀 Fitur Utama

* **Penyimpanan Lokal (LocalStorage):** Tugas kamu tidak akan hilang meskipun halaman di-refresh atau browser ditutup.
* **Desain Responsif:** Tampilan yang optimal di berbagai perangkat, mulai dari desktop hingga smartphone (Mobile First).
* **Animasi Halus:** Transisi saat menambah dan menghapus tugas untuk pengalaman pengguna yang lebih baik.
* **Custom UI:** Menggunakan ikon jam kustom dan font "Poppins" agar terlihat modern dan estetik.
* **Fitur Centang (Mark as Done):** Tandai tugas yang sudah selesai dengan satu kali klik.

---

## 🛠️ Teknologi yang Digunakan

* **HTML5:** Struktur semantik untuk aksesibilitas yang lebih baik.
* **CSS3:** Styling kustom menggunakan Flexbox, Media Queries, dan CSS Variables.
* **JavaScript (ES6):** Manipulasi DOM, Event Delegation, dan Web Storage API.
